## How to View

1. **Start HTTP Server** (if not running):
   ```bash
   cd prototype
   python3 -m http.server 8000
   ```

2. **Open Dashboard**:
   - URL: `http://localhost:8000/faculty_dashboard.html`
   - The dashboard fetches data from `dashboard_data.json`

3. **Regenerate Data** (optional):
   ```bash
   source djehuty-env/bin/activate
   python3 prototype/generate_dashboard_data.py
   ```

- **Data**: JSON file generated by Python backend
- **Backend**: Djehuty SPARQL Interface
- **No build process**: Runs directly in browser
