Problem: Fetches all datasets, processes in memory, slow for large repositories

âœ“ Suggested Fix: Leverage SPARQL Aggregation

Better Approach - Use Database for Aggregation:

# Better: Aggregate in SPARQL query
SELECT ?institution (COUNT(?dataset) AS ?count)
WHERE {
    ?dataset rdf:type djht:Dataset .
    ?dataset djht:institution_id ?institution .
}
GROUP BY ?institution
ORDER BY DESC(?count)

Benefits: Database does the counting, returns only summary, 10x faster
